DOI data model
•Two components
–Data Dictionary + DOI Application Profile Framework
•Data Dictionary
–indecs Data Dictionary.   Provides tool for precise description of entity through metadata (and mapping to other schemes).
–<indecs>  = interoperability of data in e commerce systems; adopted by ISO MPEG 21 as MPEG Rights Data Dictionary (RDD)  
•DOI-AP framework.
–Provides means of relating entities: grouping entities and expressing relationships
–An AP is a mechanism for grouping DOIs with similar properties
•DOIs, APs, DOI services:
–have many-to-many relationships: expressed through multiple resolution (handle)
–may have precise descriptions:  expressed through metadata via indecs Data Dictionary
