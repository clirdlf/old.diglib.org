DOI and existing  ISO TC46/SC9 standards
•
•Any TC46/SC9  identifier can be included using DOI syntax
–Simply use as the suffix e.g. 10.5678/ISBN-0-7645-4889-4
–
•Any TC46/SC9 metadata scheme can be mapped in DOI data tools
–
•Existing relationships e.g.
–ISBN:  good connections with DOI RAs (Nielsen BookData, Bowker); ONIX, well-formed metadata.
–ISSN:  ISSN revision under way.   Need for entity specification and relationships recognised; DOI recommended and under evaluation
–ISTC: metadata is based on indecs model (same as RDD). consortium to be the authority includes DOI RAs.
