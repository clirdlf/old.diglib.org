What is ARTstor?
  • A not-for-profit organization created by The Andrew W.  Mellon Foundation that is developing a digital library of art images to enhance scholarship, teaching and learning in the arts and associated fields
    • Image collections and descriptive data
    • Software tools to enable active use of content
    • Solely for non-commercial educational and scholarly use
3
ARTstor core functions (Green)
  • XML repository of high resolution images and rich meta data
  • www.artstor.org web site that provide tools to actively use these images
  • Browser based smart client
  • Java based Off line Image viewer/prsentation tool. Through a set of proprietary API, OIV interoperate with the ARTstor repository.

Search API into external repositories (Orange)
  • Utilizing standard OKI adapter/bridge, ARTstor should be able to search and retrieve images from institution repositories based on FEDORA, DSPACE and commercial collection system such as VITAL which is also FEDORA based.
  • ARTstor should be able to search and retrieve images from image.google.com with standard Google API.
  • ARTstor should be able to search and retrieve JSTOR art journal articles by using JSTOR XML gateway

Metasearch into ARTstor (Blue)
  • ARTstor XML gateway, return metadata and stable URL
  • Follow and compatible with JSTOR XML gateway protocol
  • ZING SRW based
  • Allow external metasearch engine such as Endeavor, Metalib etc to search into ARTstor
  • By utilizing OKI adapter can allow external java based client application such as VUE and LIONSHARE to search into ARTstor
