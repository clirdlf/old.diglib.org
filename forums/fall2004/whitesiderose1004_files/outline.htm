UCAI 101   Goal:  develop a shared cataloging resource for the vr community in order to promote the copy cataloging of image metadata
    • A tool for image catalogers
    • Focus is on work and image metadata, not delivery of digital images
    • “Art” is used loosely and meant broadly
    • A proof-of-concept, prototype database
    • The technical infrastructure for a static database of approx. 750,000 records
    • Multi-phase project
