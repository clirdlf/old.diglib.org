New Specifications
•A Metadata Steering Group was formed to review all the applicable metadata formats and document use guidelines.
•Specifications were set for the production of new digital images:
–Art, architecture, or cultural documentation images
–24-bit color or grayscale pages images
–Bitonal page images
•A new local extension of the TEI DTD for marking up Electronic Texts was developed, along with encoding guidelines.
•Specifications were set for the transition to and use of EAD (Encoded Archival Description) 2002.
