Repository Project History
•1999 – UVa Library Digital Library Research and Development reads an article about FedoraTM and begins a test implementation with Cornell University.
•2001 – DL R&D and Cornell receive a Mellon Foundation grant to develop Fedora into an open source tool.
•2003 – Fedora 1.0 is released; implementation begins on the UVa Library Digital Library Repository; a "Phase 1" prototype is internally released for review.
•2004 - DL R&D and Cornell receive a new Mellon Foundation grant to continue Fedora development; UVa Central Digital Repository is prepared for on-grounds release.
