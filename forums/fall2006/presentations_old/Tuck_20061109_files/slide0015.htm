International Developments with IIPC: Automated Smart Crawler
nAutomated Smart Crawler
nJoint project involving British Library (project lead), Library of Congress, Bibliotheque nationale de France and Internet Archive
n
nScope includes developing Heritrix crawler so that it has ability to:
nrecognize when resources have not changed from last time they were gathered
nPrioritise the order in which resources are visited and gathered
nRecognize when resources are changing more frequently and to visit them more frequently
nEnsure that new capabilities will scale to crawls of at least 100 million resources
nEnsure that web archiving community can take advantage of these new capabilities
n
