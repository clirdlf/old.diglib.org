nNutchWAX includes tools to search Web Archive Collections
n(WACs). NutchWAX is built on two other open source projects:
n
nNutch, web-search software based on Lucene Java adding Web specifics such as link-graph analysis, parsers for HTML and other formats, and a batch-oriented crawler.
n
nHadoop, a framework for running applications (such as the indexing of large scale web content) on clusters of commodity hardware. Hadoop…
nImplements a computational paradigm named map/reduce
nProvides a reliable distributed file system on the compute nodes
nHas been tested on clusters of 600 nodes, but there are reports of support for 900+ nodes for both map/reduce and file system storage.
nIA’s largest cluster for full-text indexing has been 34 nodes.
