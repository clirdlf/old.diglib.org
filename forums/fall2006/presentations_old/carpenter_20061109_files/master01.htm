•Contribution and use of open source Web archiving software

Heritrix: (http://crawler.archive.org/ ) an open source Web crawler developed by the Internet Archive with financial support from the IIPC (International Internet Preservation Consortium).
NutchWAX: (http://archive-access.sourceforge.net/projects/nutch/ ) a full text search tool (plus extensions) built on the Lucene text indexing engine that is used to search archival web content.
Wayback Machine: (http://archive-access.sourceforge.net/projects/wayback/) an open source, address-based access tool used to locate and view archived web pages developed by the Internet Archive.
