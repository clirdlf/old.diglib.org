Data Sources
  • A Reserves List is comprised of two types of data
    • Course Metadata – Managed by Schools
      • Term/year
      • Catalog Number
      • Course Name
      • Head Instructor
      • Course Instance ID


    • Reserves Lists – Managed by Libraries
      • Bibliographic Information (title, author, HOLLIS #)
      • Supporting Library
      • Lecture Date
      • Course Instance ID

    • Weekly co-lead project meetings
    • Co-develop communication protocols
    • Co-locate developers to work together
