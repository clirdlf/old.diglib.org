COUNTER is an organization that is member supported. COUNTER originated from a group called PALS which I believe stands for Publisher and Library Standards.  Members of COUNTER include libraries, publishers and aggregators and their goal is to allow credible and consistent measurement of usage statistics between vendors.
Here is the list of COUNTER usage reports that compliant vendors must supply now (when applicable).  We will cover each in detail and include an example.
The initial COP release did not provide any descriptive data within the report; therefore, if one was gathering reports for a Consortium or an individual institution, one had to be very careful in naming of files because the file itself did not declare its contents, the category, and when run.
Also to deal with changing versions, the release number is now part of the name. This will allow systems uploading reports to know what to expect.
