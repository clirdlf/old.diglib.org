Web Archives Workbench
The Web Archives Workbench will be developed as a set of open source software tools. The tools are built upon the familiar open source web server environment of Linux, Apache, and the TomCat java application server. The Discovery, Properties, Analysis, and Packager tools in turn either drive or draw data from the Heritrix harvester (open source) and the Cloudscape database (open source).

While the tools will be available as open source, OCLC plans to incorporate these tools into its Digital Archive services as well.
