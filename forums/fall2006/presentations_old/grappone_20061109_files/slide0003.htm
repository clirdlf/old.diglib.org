We are currently in phase one of the project.  Focused on proof of concept and library controlled systems.  I’ll describe a few important aspects of the project in detail and then get into a discussion of the architecture of the system.  Some content we directly ingest some content we harvest.  For example the catalog data we harvest.  It’s a 2 stage process taking the MARC records we transform them into MARC XML and then again into dublin core OAI xml.  At that point we ingest the descriptive metadata elements for the purpose of search.  Some content we simply ingest directly into the XML to store and search.
