So now let’s discuss the local cache and the XML wrapper.  It’s a simple schema which we use to wrap all of our content prior to indexing.  That content as you have seen comes in may formats but text, xml and html for the most part.  Within that local cache we have some control over display by using style sheets, rendering rules as well as store schema for XML in the cache.  In case we need some information about DTDs for style sheets and rendering rules.
