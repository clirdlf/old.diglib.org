A very brief slide about the platform we have created. All of the component parts are opensource software.    The platform we have built is written in the struts framework. Struts is an open source framework for building web applications.  The core of which is a flexible control layer based on standard technologies like Java Servlets, JavaBeans, ResourceBundles, and Extensible Markup Language (XML).

We strive to take that flexibility to the architecture as well.  Our goal is to facilitate plugging in modules such as custom indexers.  Providing users the ability to create custom interfaces to Gandhadra.  Also we worked hard to maintain flexibiltiy of the platform itself for replacing components such as the full text engine.
