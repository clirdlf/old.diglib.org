Z39.93-200X: The NISO Standardized Usage Statistics Harvesting Initiative (SUSHI) Protocol
•
•
Trial Use Period: September 20, 2006 – May 20, 2007
Abstract: This Standard defines an automated request and response model for the harvesting of electronic resource usage data utilizing a Web services framework that can replace the user-mediated collection of usage data reports. Designed to work with Project COUNTER reports, the protocol is also extensible to other types of usage reports.
