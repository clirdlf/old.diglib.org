Future Directions: Image Collection Integration

In addition to their print collection, the Workshop library has an extensive collection of images.

We will digitize and create metadata for the image collection and integrate it with the DLC, including links between images and related papers.

Technologies

  • MS Access database for metadata creation
  • Oracle database (with data imported from the MS Access data-entry tool)
  • Java-based Web application for searching metadata
