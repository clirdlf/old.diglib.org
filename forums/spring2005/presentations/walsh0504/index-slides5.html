EPrints Functionality

  • accommodates "any metadata schema"
  • allows configurable metadata to be associated with authors and documents
  • supports "configurable, extendible subject hierarchy"
  • accepts document submissions via Web interface
  • accepts documents as files, compressed bundle (e.g., .zip), or via URL reference
  • performs automated data integrity checks
  • supports an optional moderation process, which is performed via Web interface
  • provides built-in OAI data provider
