OCLC staff held these factors in mind when determining what preservation metadata elements would be needed in the first implementation of the archive:  the users’ needs, the object types, and what tools would be available

Some of the questions we asked ourselves were:
What metadata is needed for these object types? (I.e. web documents0
When is the metadata available and to whom?
How is it captured, extracted, or created (by humans, or by machine)?
How are the objects going to accessed and by whom?

We have striven for a balance of what elements are essential to preserve and maintain access to an object, what are the elements our users can practically create, and what elements the archive can extract or create.

The preservation metadata set is being developed by an OCLC team, which is informed by the working group and other digital preservation initiatives.  The preservation metadata set will be available soon for review and comments on OCLC’s website.
