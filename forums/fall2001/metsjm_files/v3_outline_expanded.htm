The METS Initiative
Create a single document format for encoding digital library objects which can fulfill roles of SIP, AIP and DIP within the OAIS reference model
Scope limited to objects comprised of text, image, audio & video files
Includes both technical component and organizational component

So what?
METS should probably a standard and relatively simple document encoding scheme for descriptive, administrative and structural metadata.
It therefore provides both a common platform for metadata exchange, and for software development for digital libraries.

Next Steps
