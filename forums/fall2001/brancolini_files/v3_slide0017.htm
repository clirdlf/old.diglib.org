Application of Knowledge
• Part-time work in some sort of DL field would be
strongly encouraged while completing coursework.
• After the above coursework is completed, an
internship would be completed involving managing a
small-scale project or planning a pilot for a larger
project. This workplace experience is essential in
order to tie all the coursework into real-word
applications. The intern would also learn about things
that are hard to teach in a class -- workflow issues,
organizational politics, etc.
