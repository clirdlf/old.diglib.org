And as Norman talked about, Multiple Resolution is a key development.  CrossRef is looking at many areas where this could add value for journals.  Tomorrow’s IDF session was billed as using the DOI for commercials transactions and said “turn every DOI into a sales opportunity” – from CrossRef’s perspective, this is a little too narrow.  Multiple resolution will make linking more sophisticated and can be used in many different ways – mirrors copies of articles, different versions or rights information about an item.
