This is an object record display for one of the pamphlets in the May Anti-Slavery Collection.  We’re planning to import the pamphlet records for this collection into a single repository, which, for a relatively homogeneous repository such as this one, opens up some opportunities with regard to display.  The rather plain display you see here is a carrier for the metadata and little more.  But because all of the contents of this repository are pamphlets, and as such are items of relatively small granularity within the overall scheme of the collections we are offering, we can use the display to convey those characteristics of format and granularity in words, icons, or both.
