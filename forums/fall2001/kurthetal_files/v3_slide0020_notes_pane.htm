Notice the wording in the red bar: “Parents of Evening Grosbeak.”  This gives users some indication of where Evening Grosbeak belongs, but there is no sense of context, of how this object fits with the bigger picture.

So how can we break out of this “flatland?”  How can we make the data speak for itself?  How do we convey complex relationships which extend beyond two-dimensional space and indeed can mutate depending on user manipulation?  Further, if in future we offer a customizable interface, a sort of “My ENCompass,” one that will create a profile based on search history and habits, how do we help users make sense of relationships between “things” as they change?
