ACLS History E-Book Project
•SPO provides publishing system for page image and XML encoded book, image viewer/database, search engine, authentication, MARC cataloging, scanning, statistical reporting.
•
•Project includes “backlist” books which are previously published titles selected by scholars. Backlist books delivered as page image scans (TIFF to GIF). Searching is on uncorrected OCR text.
•
•Project also includes “frontlist” books which are new books encoded in XML. Frontlist books, are encoded in XML and delivered in HTML for display in standard web browser.
•
•Production, design, proofing stages - use XSLT to transform XML
•
•Specs, DTD, XSLT, templates, proofing guidelines available on our web site. Used by publishers to create files for submission.
