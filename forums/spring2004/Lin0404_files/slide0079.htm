DLF April 2004
ACLS History E-Book Project
Concluding thoughts, some challenges
• Built framework and system, interesting new projects being submitted.
•
• Workflow developed. Specs, DTD, scripts in place, publisher and vendor relations set up.
•
• Some challenges:
• For SPO managing large digital collection different than working on new e-books one at a time.
• Software tools for writing, editing, production, still need development.
• Lack of protocols for linking, citing, labeling.
• URL based linking not stable.
• Most collections do not offer clear permanent IDs nor do they offer commitments to permanence in distributing digital object.
• New e-books are interwoven into network of digital collections - journal articles, reviews, other monographs (historiography), image collections, transcriptions, etc.
•
• Need to build good infrastructure and protocols.
