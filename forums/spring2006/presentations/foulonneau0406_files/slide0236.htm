Concluding Thoughts
nConclusions
nSet descriptions are recommended
nDLF/NSDL best practices documents identify the information needs for selective harvesting
nSet descriptions and collection descriptions are not the same
nSome harvesters also want collection level descriptions to add context to items, particularly in the cultural heritage arena.
nUsing set descriptions to convey collection level descriptions may raise difficulties in the long run
nNo identification scheme for collections
nSince set description records have no datestamp, there is no trigger for automatic update
