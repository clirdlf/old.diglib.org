Sets vs. Collections
nA set is a "collection" of records
nHarvesters want information about the underlying content
nScope (description), subject terms, coverage, type of resource, rights for content, link to online presentation of collection
nand about the set of records
nWhat entity is responsible for the set?
nIs the set being added to or updated routinely?
nE.g., because more is being described /digitized
nWhat quality of cataloging to expect?
nWhat is the native format for the records?
