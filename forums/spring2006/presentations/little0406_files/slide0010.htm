Data model: FRBR
•Data model emerged from research into the Functional Requirements for Bibliographic Records (FRBR)
•FRBR model collapsed into 2 levels
•Work
•Describes an analog object that has been digitized or a born digital object
•Item
•Describes the digital representation of the object
•Disparate file types (PDF, jpg, xml, mp3, wav, etc.)
