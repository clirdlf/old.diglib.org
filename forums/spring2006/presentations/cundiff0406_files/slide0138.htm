METS Profile for Bibliographic Records
The Library of Congress METS Profile for Bibliographic Records specifies how METS documents representing bibliographic records within Library of  Congress digital library projects should be encoded. This profile is intended to be used in cases where there is a need to include bibliographic records for items that have not yet been digitized in a digital library project. The profile can also be used in cases where there is a need to associate a partially digitized item with the bibliographic data (e.g. a bibliographic record for a book where the just the book cover or the pages of the table of contents have been digitized).
