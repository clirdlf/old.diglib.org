CD Profile: Design Goals
•
•Express the logical structure of the object using MODS
•Express the physical structure of the object using METS
•Provide a correlation between the two hierarchical structures
•Allow for a reasonable amount of variation (i.e. some options in the form of occurrence indicators: mandatory, optional, repeatable, etc.)
•But not infinite variation (leading to too much complexity that defeats the purpose).
•Balanced, “90% solution”
•Develop profile methodology that can be used for other profiles
