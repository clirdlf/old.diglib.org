DAK - Metadata
§Descriptive Metadata
–For textual materials bibliographic metadata retrieved from the  BA ILS where books are catalogued in MARC21.
–For multimedia objects VRA fields (Title, Creator(s), Date(s), mateial, technique, Keywords, Description, Dimensions, Location, etc)
§Administrative Metadata
–Information about the batches, which are received from the owner.
–Record document condition, page count, etc.
–Allocate the document to a Collection
–Record of completion of each phase with specialist name and date.
–Information regarding the intellectual property rights
–Authentication and authorization data
§Technical Metadata
–Digital Content Metadata: image resolution, library font, profile, versions
–Archiving Metadata
–Publishing Metadata
