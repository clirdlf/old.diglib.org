This is a workflow schema that can be modified for every type of material and scanning device. Depends on the nature of the item, some of the activities can be batched, such as color correction, de-skewing, or geometrical correction. For example, rare archival documents will call for more detailed or individual treatment.

This specific example is for a bound book scanned with the DigiBook.
Splitting is an optional step, when we deal with the request for both double page images and single page images.
And finally, the creation of derivatives, which is obviously a batch process.
