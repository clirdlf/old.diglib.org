Dealing with a lot of data is not something that one would want to do in paper based environment. Although we still use some paper based documentation forms in our daily work, we move more and more towards online tracking and statistics generation. Our main tool is SharePoint, by Microsoft. Job assigning for student technicians is done via this tool, as well as listings of projects and gathering of performance statistics.
