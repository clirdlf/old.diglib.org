Trends in Data Needs
  • Increased emphasis on secondary data analysis in research
  • Increased use of data in classroom
  • Demand for targeted data sharing
  • Data archiving needs
4
DEWI Goals
  • Create an easy-to-use, platform independent, one-stop-shop for data discovery and extraction that allowed data users to:
    • Search or browse variables
    • View other important data information
    • Extract and download their data
    • Focus on data analysis
  • Restrict access to DEWI content and functionality.
  • Do not become a system, web, or database administrator !
