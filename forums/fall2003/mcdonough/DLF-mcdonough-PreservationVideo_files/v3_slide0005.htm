Digital Video Basics
Why not 4:4:4 sampling?
n 720 x 486 resolution = 349,920 pixels per frame
n 349,920 pixels x 10 bits/sample x 3 samples/pixel = 10,497,600
bits per frame
n 10,497,600 bits/frame X 29.97 frames/second = 314,613,072
bits per second
n 314,613,072 bps x 3600 seconds = ~141.58 GB/hour
n For 1920x1080 HDTV, more like 840 GB/hour
4:2:2 sampling drops that rate by a third with almost
no perceptible difference in quality.  4:2:0 and 4:1:1
drop it in half.
