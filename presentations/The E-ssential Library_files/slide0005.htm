The New Realities
•It is not a “web world” only any more – users are beginning to expect ebook, PDA, print-on-demand, wireless.
•Content needs to conform to user needs, and not vice versa.
•Libraries are losing users to Google, Amazon, and Blackboard – we need to reclaim them
•“If we build it we will come” is too often true of what we produce – web services for the ideal user (us)
