Conclusions
nWe need the data that resides on a publishers’ site to be much easier for us to re-shape for local delivery. Publishers need to help their customers -- libraries – market the publishers’ data to the faculty and students we serve. When we all have much the same content, better service is what distinguishes a library.
nThe standardized and malleable document – that which survives is likely to be that which is easily migrated, and that which supports a sufficient range of re-use to keep it economically viable.
nWe need to move from data silos defined by publisher to data that can be deeply shared across publishers.
