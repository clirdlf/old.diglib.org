Image State:
•
Basic and Intermediate - imaging done in a manner to facilitate output, adjusted for either display or print - all images to be positive (i.e. negatives inverted to be a positive image)
•
Advanced and Future - imaging done in a manner that relates the digital image back to the originals - all images original tone orientation (i.e. negatives remain negative in master files) - images will need to be adjusted for use, for both display and print
